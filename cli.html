Command-line interface (CLI) and shell scripting

mkdir
- make a new directory within the current directory
cd
- change the current directory to another
pwd
- check your current working directory
touch
- creating new files in your current directory and also to modify file timestamps
cat
- print the content of a file to the command line screen
echo
- print characters to the screen
rm
- delete specified files in the current directory or the directory specified
rmdir
- delete a specified empty directory.
cp
- copy the content of one file to another file or duplicate a file. When copying to an existing file, it overwrites the previous text to the new one
mv
- cutting the content of one file to another. It can also be used to rename a file.
grep
- (Global Regular Expression Print): This is used to search for strings within a specified file. Use grep as follows; grep [options] [string] [specified file]
