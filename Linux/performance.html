Performance Tuning

Analyze bottlenecks and optimize hardware resources for peak efficiency.

Memory & CPU vmstat

Reports virtual memory statistics, including processes, memory, paging, block IO, and CPU activity.

vmstat 1 5
Disk I/O iostat

Crucial for identifying disk bottlenecks. Shows how much data is being read/written to your drives.

iostat -xz 1
Load Average uptime

The quickest way to check "Load Average." It shows if your CPU is over-saturated over 1, 5, and 15 minute intervals.

uptime
Kernel Tuning sysctl

Modify kernel parameters at runtime. Used by DevOps to tune network stacks and file limits.

sudo sysctl -p
Hardware info lscpu

Provides detailed information about the CPU architecture, cores, threads, and cache sizes.

lscpu
Storage capacity df -h

Reports the amount of disk space used and available on file systems. Essential for preventing "Disk Full" crashes.

df -h
