Process Management

Monitor, control, and optimize running applications and system resources.

top Live View

The classic real-time system monitor. Displays CPU usage, memory, and a list of running tasks.

top

Press 'q' to exit the live view.

ps Snapshot

Shows a static "Process Status" snapshot. Use aux to see every process running on the system.

ps aux | grep nginx
kill Critical

Used to terminate a process using its PID (Process ID). The -9 flag forces a shutdown.

kill -9 1234
systemctl Services

The modern way to manage background services (daemons). Use it to start, stop, or restart apps.

sudo systemctl restart docker

Essential for managing web servers and databases.

nice / renice

Changes the priority of a process. "Niceness" ranges from -20 (highest priority) to 19 (lowest).

nice -n 10 ./backup.sh
free

Displays the total amount of free and used physical memory (RAM) and swap memory in the system.

free -h

The -h flag makes the output "Human Readable".
