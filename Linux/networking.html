Networking & Firewall

Manage ports, analyze traffic, and secure your server's perimeter.

INTERFACE ip addr

The modern replacement for ifconfig. Shows all network interfaces and their assigned IP addresses.

ip addr show
CONNECTIONS netstat

Displays active network connections and listening ports. Vital for checking if a web server is up.

netstat -tuln
SECURE ACCESS ssh

Secure Shell. Used to log into remote servers encrypted over an insecure network.

ssh user@server-ip
FIREWALL ufw

Uncomplicated Firewall. A beginner-friendly way to manage IPTables rules and block ports.

sudo ufw allow 80/tcp
HTTP TOOLS curl / wget

Command-line tools to transfer data. curl is for requests; wget is for downloading files.

curl -I google.com
DNS nslookup

Used to query DNS records. Use this to verify if your domain points to the correct IP address.

nslookup mywebsite.com
