Package Management

Software in Linux is managed through repositories. Here is how you handle installations on the most common distributions.

Debian / Ubuntu / Kali APT (Advanced Package Tool)

The standard for Debian-based systems. Known for its ease of use and powerful dependency resolution.

Update Repositories sudo apt update
Install a Package sudo apt install [package_name]
Remove Software sudo apt remove [package_name]
DevOps Tip: Use apt-get install -y in your Dockerfiles to auto-confirm installations.
RHEL / CentOS / Fedora YUM / DNF

Yellowdog Updater, Modified (YUM) and its successor DNF are used in enterprise-grade Red Hat distributions.

Refresh Metadata sudo dnf check-update
Install a Package sudo dnf install [package_name]
Clean Cache sudo dnf clean all
Fact: On modern systems, yum is usually just a symbolic link to dnf.
