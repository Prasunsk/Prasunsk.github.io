Linux Command Encyclopedia

Every essential command you need to navigate and manage a Linux environment.

mkdir

Make Directory: Used to create a new folder. It can create multiple directories at once or nested paths.

mkdir -p project/src/components

Tip: The -p flag creates parent directories if they don't exist.

cd

Change Directory: The primary tool for navigating the file system. Moves your terminal session to a new location.

cd /var/log/nginx

Tip: Use "cd -" to jump back to your previous directory.

pwd

Print Working Directory: Tells you exactly where you are in the system by showing the full absolute path.

pwd
touch

Create File: Creates a new empty file instantly. It is also used to update the access/modification time of a file.

touch index.html style.css
cat

Concatenate: Read the entire contents of a file and output it directly to your terminal screen.

cat config.yaml
echo

Print Text: Outputs strings or variables to the terminal. Frequently used to write text into files using redirection.

echo "API_KEY=123" >> .env
rm

Remove: Deletes files or directories. This action is permanent and cannot be undone via the terminal.

rm -rf ./old_project

Warning: Be extremely careful with the -rf flags.

rmdir

Remove Directory: A safer way to delete directories; it only works if the directory is completely empty.

rmdir empty_logs_folder
cp

Copy: Duplicates files or folders. Use the -r flag to copy a folder and everything inside of it.

cp main.js backup_main.js
mv

Move/Rename: Relocates a file to a new path or changes its name. Unlike copy, it doesn't leave the original behind.

mv script.sh ./scripts/
grep

Search: Filters through text or files to find lines that match a specific pattern or keyword.

grep -i "error" server.log

Tip: Use -i to ignore case sensitivity.

ping

Network Test: Checks if a remote server is reachable and measures the time it takes for data to travel.

ping google.com
unzip

Extract: Uncompresses .zip files to retrieve the files stored inside. Essential for downloading packages.

unzip deployment.zip
whoami

Identity: Displays the name of the user currently logged into the terminal session.

whoami
